# Project Overview

This is a starter project for [Astro](https://astro.build), a web framework for building fast, content-focused websites. It uses [Tailwind CSS](https://tailwindcss.com/) for styling.

The project is structured as follows:

- `src/pages/index.astro`: The main page of the site.
- `src/components/Welcome.astro`: A component used on the main page.
- `src/layouts/Layout.astro`: The main layout for the site.
- `astro.config.mjs`: The Astro configuration file.
- `package.json`: The project's dependencies and scripts.

# Building and Running

To build and run the project, use the following commands:

| Command | Action |
| :--- | :--- |
| `npm install` | Installs dependencies |
| `npm run dev` | Starts local dev server at `localhost:4321` |
| `npm run build` | Build your production site to `./dist/` |
| `npm run preview` | Preview your build locally, before deploying |

# Development Conventions

The project uses the standard Astro project structure and conventions. It also uses Tailwind CSS for styling.
